Documentación con Open API - Swagger

En Laravel se se puede utilizar un paquete como L5-Swagger para documentar el código. Se emplean anotaciones del tipo:

/** * @OA\Get( * path="/api/pacientes", * summary="Listar pacientes", * tags={"Pacientes"}, * @OA\Response( * response=200, * description="Listado de pacientes" * ) * ) */ public function index() { ... }

Mediante el comando "php artisan l5-swagger:generate", se genera un archivo json que compila las anotaciones que encuentra en nuestro código con la etiqueta "@OA"
