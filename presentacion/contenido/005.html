Documentación con Open API - Swagger

En Laravel se se puede utilizar un paquete como L5-Swagger para documentar el código. Se emplean anotaciones con la etiqueta "@OA"

Mediante el comando "php artisan l5-swagger:generate", se genera un archivo json que compila las anotaciones que encuentra en nuestro código con esta etiqueta y la muestra ordenadas, con la posibilidad de comprobar las rutas
